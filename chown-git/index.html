Git 忽略文件目录权限或文件拥有者的改变

Git
2020-06-13

git 中可以加入忽略文件权限的配置，在 git 根目录运行如下命令：

当前版本库忽略文件权限

git config core.filemode false

复制所有版本库忽略文件权限

git config --global core.fileMode false

这样就设置了忽略文件权限。

查看 git 的配置文件：

cat .git/config

这时候再更新代码就 OK 了。
